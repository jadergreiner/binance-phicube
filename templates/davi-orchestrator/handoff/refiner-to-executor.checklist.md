# Refiner -> Executor Handoff Checklist

- [ ] Refiner spec exists and is complete.
- [ ] Scope is explicit (in/out).
- [ ] Tasks are concrete and ordered.
- [ ] Acceptance criteria are testable.
- [ ] Constraints and non-goals are explicit.
- [ ] Risks and mitigations are listed.
- [ ] Human approval is registered.
- [ ] No unresolved critical ambiguity remains.
- [ ] Artifact lint gate PASS for changed artifacts.

## Handoff Decision

- [ ] Pass to Executor
- [ ] Return to Refiner
