# Skills Catalog Template: Architecture

## Mandatory Skills (minimum set)

1. System Mapping

- Purpose: identify impacted modules and boundaries.
- Trigger: change spans multiple components.

1. Design Trade-off Analysis

- Purpose: compare options and justify selected approach.
- Trigger: more than one viable architecture path exists.

1. Task Slicing

- Purpose: decompose implementation into executable slices.
- Trigger: executor needs clear delivery sequence.

## Optional Skills

1. Performance Strategy

- Purpose: capture latency/throughput implications.

1. Failure Mode Analysis

- Purpose: model technical failure scenarios and mitigations.
