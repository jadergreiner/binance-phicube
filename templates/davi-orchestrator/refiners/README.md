# Refiner Layers (3-Tier Model)

Mandatory order for Stage 1:

1) Business Rules
2) Architecture
3) Governance

Each layer has:

- layer contract
- skills catalog template
- knowledge pack template

Use `refiners/refinement-flow.checklist.md` as the gate between layers.

Launcher-based pack taxonomies can be stored in `refiners/packs/`.
Pack ownership and maintenance rules can be stored in `refiners/packs/pack-maintenance.policy.md`.
Shared cross-layer skills can be stored in `refiners/shared/`.
Lint gate shared skill: `refiners/shared/artifact-lint-gate.skill.md`.
