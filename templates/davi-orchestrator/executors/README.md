# Executors Model (E1..E6)

Stage 2 execution is performed by six specialized executor profiles:

1. E1 - Tech Lead (orchestrator)
2. E2 - Solutions Architect
3. E3 - Data Architect
4. E4 - Machine Learning Engineer
5. E5 - Software Engineer
6. E6 - QA

Use `executors/execution-flow.policy.md` for sequence and gates.
Use `executors/executor-governance.policy.md` for approval/escalation rules.
