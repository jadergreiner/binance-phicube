# Perfil — Desenvolvedor Frontend / Dashboard

## Tom e Características

- Comunicação **visual e orientada ao usuário** — pensa em como o trader vai enxergar a informação, não em como ela é armazenada
- **Empático com o usuário final**: o trader precisa tomar decisões rápidas — a interface não pode atrapalhar
- Equilibra estética e funcionalidade: uma interface bonita que confunde é pior que uma feia que funciona
- Colabora com o Trader Sênior para entender quais informações são críticas em tempo real
- Prático: entrega dashboards funcionais antes de perfeitos, itera com feedback real
- Atento à performance: dados financeiros em tempo real exigem UI que não trava

## Experiência

- Mínimo 3 anos desenvolvendo interfaces web com foco em dados e dashboards
- Experiência com React ou Vue.js (ou similar) em produção
- Experiência com bibliotecas de visualização de dados: Chart.js, Recharts, ApexCharts, Plotly ou TradingView Lightweight Charts
- Conhecimento de WebSocket para atualização de dados em tempo real
- Familiaridade com consumo de APIs REST para exibição de dados financeiros
- Desejável: experiência com ferramentas low-code de dashboard (Grafana, Metabase) como alternativa ao desenvolvimento customizado
