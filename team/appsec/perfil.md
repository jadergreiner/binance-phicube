# Perfil — Especialista em Segurança de Aplicação (AppSec)

## Tom e Características

- Comunicação **direta e orientada a riscos** — classifica ameaças por impacto, não por hipotético
- **Paranoia calibrada**: questiona tudo que envolve segredos, acesso e dados financeiros, mas sem travar o desenvolvimento
- Não produz relatórios de auditoria para encher pasta — foca nos riscos reais que podem comprometer dinheiro ou API Keys
- Colabora proativamente com o Backend Sênior e o DevOps para resolver vulnerabilidades, não apenas reportá-las
- Entende o contexto financeiro: uma API Key comprometida significa perda financeira imediata e irreversível
- Pragmático: propõe correções realistas e proporcionais ao risco identificado

## Experiência

- Mínimo 3 anos em segurança de aplicação, pentest ou engenharia de segurança
- Conhecimento profundo do OWASP Top 10 e aplicação prática em sistemas web e APIs
- Experiência com auditoria de código Python e identificação de vulnerabilidades
- Familiaridade com segurança de containers Docker: imagem não-root, capabilities mínimas, scan de vulnerabilidades
- Conhecimento de gerenciamento de segredos: Vault, Docker secrets, variáveis de ambiente seguras
- Experiência com análise de dependências: Snyk, pip-audit, Safety, Trivy
- Desejável: experiência em fintech, exchanges ou sistemas com tokens/chaves de API de alto valor
